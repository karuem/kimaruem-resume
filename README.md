## 🌱 학습 및 목표 (Learning Goals)
> “배운 것을 빠르게 실험하고, 실험한 것을 기록하며 성장한다.”

### 🧱 백엔드 아키텍처 및 서버 개발
현재 Java와 Python을 중심으로 백엔드 개발의 기본기를 쌓고 있습니다.
프로그램이 실제로 어떻게 실행되는지 핵심 개념을 익히며 안정적으로 동작하는 코드를 설계하는 방법을 연습하고 있습니다.

앞으로는 데이터 모델링과 서비스 구조 설계까지 학습 범위를 넓혀
유지보수가 쉬운 백엔드 구조를 설계할 수 있는 개발자를 목표로 하고 있습니다.
---

### ☁️ 클라우드 환경 및 배포 자동화
 AWS EC2, S3 등을 활용하여 서버를 배포하고 관리하는 과정을 실습할 계획입니다.
이를 통해 로컬 환경을 넘어 실제 서비스 환경에서의 배포 흐름을 익히고,
CI/CD와 컨테이너(Docker) 개념을 학습하여 자동화된 배포 방식의 이해도를 높이고자 합니다.  

---

### 🧩 데이터 분석 및 시스템 관측
데이터의 흐름을 시각적으로 표현하고 분석하는 과정에 관심이 많습니다.  
`matplotlib`과 `NumPy`를 활용하여 PCM 신호 분석 결과를 시각화했으며,  
이를 바탕으로 향후 `Prometheus`, `Grafana` 등의 모니터링 도구를 학습하여  
시스템의 성능과 신뢰성을 정량적으로 측정하고 개선하는 능력을 기르고자 합니다.  

---

### 💡 문제 해결과 성장 방향

**문제를 근본적으로 이해하고 해결하는 개발자**를 목표로 하고 있습니다.  

코드 리뷰와 GitHub 협업을 통해 논리적 사고를 다듬고 있으며,  
작은 프로젝트라도 설계·구현·문서화의 전 과정을 스스로 관리하면서  
개발의 전체 흐름을 익히고 있습니다.  
앞으로는 오픈소스 기여와 팀 프로젝트를 통해  
더 넓은 시야로 문제를 바라보는 능력을 키워갈 것입니다.
