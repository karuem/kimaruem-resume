### 🧱 백엔드 아키텍처 및 서버 개발
Java와 Python을 중심으로 한 백엔드 아키텍처 설계 능력을 기르는 것을 목표로 하고 있습니다.  
Spring Framework와 FastAPI를 활용하여 RESTful API를 설계하고, 데이터베이스와의 연동 구조를 효율적으로 설계하는 방법을 익히고 있습니다.  
이 과정에서 단순한 코드 구현을 넘어, **동적 메모리 관리와 파일 입출력, ORM 기반 데이터 모델링**을 통해 서비스의 안정성과 확장성을 고려한 개발을 지향합니다.  



### ☁️ 클라우드 환경 및 배포 자동화
안정적인 서버 운영 환경을 구축하기 위해 AWS EC2, S3 등 클라우드 서비스를 활용해
서버를 직접 배포하고 관리하는 과정을 학습하고자 합니다.
이를 통해 로컬 개발 환경을 넘어, 실제 서비스 운영에 필요한 배포 자동화(CI/CD) 와
안정적인 서버 인프라 구성 능력을 갖추는 것을 목표로 하고 있습니다.
또한, Docker와 Kubernetes를 이용한 컨테이너 기반 배포 방식을 익혀
확장성과 효율성을 고려한 백엔드 개발자로 성장하고자 합니다.  



### 🧩 데이터 분석 및 시스템 관측
데이터의 흐름을 시각적으로 표현하고 분석하는 과정에 관심이 많습니다.  
matplotlib과 NumPy를 활용하여 PCM 신호 분석 결과를 시각화했으며,  
이 경험을 바탕으로 향후 Prometheus나 Grafana 같은 모니터링 도구를 활용해  
시스템의 성능과 신뢰성을 정량적으로 측정하고 개선하는 능력을 기르고자 합니다.
